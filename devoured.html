Devoured: A Data Visualisation Project

Year 4 University Project

The project

‘Devoured’ is a data visualisation project that examines my daily kilojoule intake during a single week. This project looks at the correlation between my kilojoule intake, and a multitude of variables. The wire-bound book format ensures each graphic is seen individually, and doesn’t overwhelm the reader. This project visualises my kilojoule intake from three perspectives. These are: the kilojoule intake from each daily meal; the percentage of kilojoules consumed each day; and the time of day when different meals and food groups were consumed. A number of different data visualisation techniques are used, including bar, pie, bubble charts and radar charts, and photographs.
