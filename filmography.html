Filmography

Interactive Film Database

Year 4 University Project

The project

Filmography is an interactive site that visualises films from 1902 to 2017 in a timeline. The intention behind Filmography is to encourage users to discover films in a new and novel way. It’s best used prior to watching a film, as it allows users to discover films they may not have seen otherwise, and prompts them to expand their film knowledge. Filmography’s goal is to give users an immersive, interactive experience that is unique to them. Films can be found by filtering film genres and navigating the timeline. Rather than searching for a specific film or director, this process encourages users to take a playful, exploratory approach.

Site design

The team

This project was completed as a team. The team includes myself, Grace Searson and Stephanie Brink. As web designers, each of us was able to design, create mark-up, style, write code and develop the user experience. To help structure the project and delegate tasks, we used the Scrum Framework. Within this framework my role was Scrum Master. We used The Movie Database API to source the data.
